# Deployment Requirements

## MetaTrader5 Issue Resolution

MetaTrader5 is not available on most cloud platforms (Render, Railway, Heroku). Here's how to handle this:

### Option 1: Use Simulation Mode (Recommended for Cloud)
The bot automatically falls back to simulation mode when MT5 is unavailable - this works perfectly!

### Option 2: VPS with MT5
If you need live data, deploy on a VPS with Windows and MT5 installed.

### Option 3: Modify Requirements for Cloud
Create a cloud-specific requirements file without MT5:

```bash
# requirements-cloud.txt
python-telegram-bot>=20.0
pandas>=2.0.0
numpy>=1.24.0
pandas-ta>=0.3.14b0
python-dotenv>=1.0.0
aiohttp>=3.8.0
```
